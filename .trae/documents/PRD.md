# 摸鱼计时器 - 产品需求文档

## 1. 产品概述

一款专为打工人设计的桌面计时器应用，帮助用户记录工作日的"摸鱼"时间和加班时间，自动计算摸鱼赚取的"工资"和加班扣减的费用，最终给出当日净收益。

目标用户：希望趣味性记录工作状态、统计时间价值的上班族。

## 2. 核心功能

### 2.1 基础设置
- **薪资录入**：输入月薪或年薪，自动换算时薪
- **上下班时间**：设置标准工作时段

### 2.2 摸鱼计时
- **摸鱼按钮**：一键开始/结束摸鱼计时
- **实时计时**：显示当前摸鱼时长
- **收益计算**：基于时薪实时计算摸鱼赚取金额

### 2.3 加班录入
- **加班时间录入**：下班后可录入加班时长
- **加班费计算**：按1.5倍时薪计算加班收入

### 2.4 统计报表
- **摸鱼费用**：正值，显示摸鱼赚取金额
- **加班费用**：负值，显示加班应得收入
- **净收益**：摸鱼费用 + 加班费用的总和

## 3. 核心流程

```mermaid
flowchart TD
    A[录入薪资信息] --> B[设置上下班时间]
    B --> C{上班中}
    C -->|摸鱼开始--> D[点击摸鱼按钮]
    D --> E[开始计时]
    E --> F[点击结束摸鱼]
    F --> G[计算摸鱼收益]
    G --> C
    C -->|下班--> H[录入加班时间]
    H --> I[计算加班费用]
    I --> J[统计总览]
    J --> K[摸鱼费用 + 加班费用 = 净收益]
```

## 4. 用户界面设计

### 4.1 设计风格
- **主题**：现代极简 + 趣味性元素
- **色调**：主色采用温暖的橙黄色（代表摸鱼的轻松感），辅以深蓝灰色（代表工作的严肃感）
- **字体**：等宽字体显示数字，强调现代感
- **布局**：单页应用，垂直布局，操作便捷

### 4.2 界面元素
| 模块 | 元素 | 说明 |
|------|------|------|
| 薪资设置 | 输入框 + 单位切换 | 支持月薪/年薪切换 |
| 时间设置 | 时间选择器 | 开始/结束时间 |
| 摸鱼计时 | 大按钮 + 计时显示 | 突出的"摸鱼"按钮 |
| 加班录入 | 数字输入框 | 加班小时数 |
| 统计面板 | 卡片展示 | 三栏：摸鱼费用/加班费用/净收益 |

### 4.3 交互反馈
- 摸鱼开始时：按钮变色 + 轻微震动效果
- 计时进行中：动态数字变化
- 统计完成：数字飞入动画
